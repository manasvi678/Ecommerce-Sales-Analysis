# E-Commerce Sales Analysis Dashboard

## 🎯 Project Objective

To design and develop an interactive and insightful Sales Analysis Dashboard using SQL and Power BI. The project focuses on analyzing e-commerce sales data to uncover trends in revenue, product performance, customer behavior, and regional insights for better business decision-making.

---

## ✅ Features

📅 **Monthly Sales Trend Analysis**
Visualizes sales over time to identify growth patterns and peak sales periods.

📦 **Category & Product Performance**
Analyzes which product categories and products generate the highest revenue.

🌍 **Region-wise Sales Insights**
Provides insights into sales distribution across different regions.

🏆 **Top Products Analysis**
Identifies the top-performing products based on sales.

🎛️ **Interactive Filters (Slicers)**
Allows filtering by Category, Region, and Order Date for dynamic analysis.

📊 **Clean and Interactive Dashboard**
Includes bar charts, line charts, and slicers for easy understanding.

---

## 🧹 Data Cleaning & Preparation

* Checked for missing (NULL) values in key columns
* Verified and handled duplicate records
* Ensured data consistency for accurate analysis
* Structured data using SQL before visualization

---

## 🛠️ Tools Used

* **MySQL** – Data storage, cleaning, and analysis
* **Power BI Desktop** – Dashboard creation and visualization
* **CSV Dataset (Superstore.csv)** – Source data
* **Microsoft Excel (optional)** – Initial data inspection

## 🚀 How to Use

1. Import the dataset into MySQL and perform data cleaning.
2. Connect Power BI to MySQL database.
3. Load the **superstore** table.
4. Explore the dashboard visuals:

   * Monthly sales trends
   * Category performance
   * Top products
   * Regional insights
5. Use slicers to filter data dynamically.
